Nota de Seguridad Importante: Este archivo muestra las contraseñas en texto plano con fines demostrativos. En una aplicación real, **nunca** debes almacenar contraseñas de esta manera. Las contraseñas siempre deben ser almacenadas de forma segura (por ejemplo, con **hashing** y **salting**) en una base de datos en un servidor seguro, no en el almacenamiento local del navegador. `localStorage` no es un lugar seguro para datos sensibles de usuario.
